# Javascript Game
A simple javascript game.

## Installation
```sh copy
npm install https://github.com/Nurutomo/js-games.git
```

## Build Setup
```sh copy
npm install
```

### Building
```sh copy
npm run build
```

### Structure
- `src/` source files (`index.ts`, `utils/*`, `games/*`)
- `src/test/` tests files (`*.test.ts`)
- `dist/` transpiled output

## Testing
```sh copy
npm test
```
